# FRAME Lab Reports

Put saved comparison outputs here.

Good report types:
- one-candidate vs many fixtures
- many candidates vs one fixture
- cross-fixture average summaries
- repeated friction patterns that suggest a FRAME change

Suggested naming:
- `YYYY-MM-DD-<candidate>-core7-summary.md`
- `YYYY-MM-DD-<fixture>-candidate-comparison.md`

Rule of thumb:
raw per-fixture judgments live in scoring scorecards
cross-score synthesis lives here
