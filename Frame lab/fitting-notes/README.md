# FRAME Lab Fitting Notes

Use this folder for cross-fixture lessons that survive more than one pass.

Examples:
- repeated confusion between two candidate parts
- fields that keep going stale
- semantics that should probably stay extensions
- family pressures that keep showing up across multiple repos

Rule:
if it is just one fixture's local judgment, keep it in that scorecard
if it keeps repeating across fixtures, promote it into a note here
