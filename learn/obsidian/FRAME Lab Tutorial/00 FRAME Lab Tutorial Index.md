# FRAME Lab Tutorial Index

This tutorial explains what FRAME Lab is, what the current 0.2.0 line looks like, and how to think about the whole thing without getting lost in files, schemas, scores, and agents.

Frame Lab is the testing ground for FRAME.

If FRAME is trying to become a portable project-context standard, Frame Lab is where we ask:

> Does this shape actually survive real repos, or does it only sound good in our heads?

## One-sentence version

Frame Lab takes real projects, tries to describe them using the current FRAME line, scores where that description works or breaks, then adjusts FRAME carefully based on repeated evidence.

## Important warning

Frame Lab is not about making a beautiful document system.
It is about pressure-testing whether FRAME can help agents and developers understand a project without drowning in random context.
